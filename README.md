# Traffic Light Controller using Verilog

## About
This is a simple traffic light controller I designed using Verilog. 
It is based on a finite state machine (FSM) and controls signals for a main road and a side road.

## How it works
There are 4 states:
- Main green, side red
- Main yellow, side red
- Main red, side green
- Main red, side yellow

Each state runs for a fixed number of clock cycles using a counter.

## What I learned
- FSM design in Verilog
- Writing sequential and combinational logic
- Handling counters and state transitions
